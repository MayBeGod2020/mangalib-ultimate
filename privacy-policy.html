Privacy Policy

Last updated: May 2025

This Privacy Policy describes how the Mangalib Ultimate Helper Chrome extension ("Extension") handles user data.

1. Data We Do NOT Collect

The Extension does not collect, transmit, or store any personal data on our servers. We have no backend that receives your data.

2. Data Stored Locally

The following data is stored exclusively in your browser using chrome.storage.sync and chrome.storage.local:

  • Extension settings (toggles, preferences, theme)
  • Your AI provider API key (entered voluntarily by the user)
  • Moderator watchlist and cached user data

This data never leaves your browser except as described below.

3. External Requests Made by the Extension

3.1 AI Providers (optional, user-initiated)

If you enable the AI analysis feature and enter an API key, the Extension will send comment text to your chosen AI provider for moderation analysis. Supported providers:

  • DeepSeek (api.deepseek.com)
  • OpenAI (api.openai.com)
  • Google Gemini (generativelanguage.googleapis.com)
  • Anthropic Claude (api.anthropic.com)
  • Alibaba Qwen (dashscope.aliyuncs.com)
  • xAI Grok (api.x.ai)
  • Mistral AI (api.mistral.ai)

The API key is stored only in your local browser and is sent only to the provider you select. We do not receive or store this key. Refer to each provider's own privacy policy for how they handle the data.

3.2 Lib-family Site API (api.cdnlibs.org)

The Extension queries public user profile data (level, roles, ban history) from the site's API when you hover over a username. This uses the same API that the site itself uses and is subject to the site's own terms of service.

3.3 Firebase Realtime Database

The moderator dashboard feature stores online-status timestamps and team announcements in a Firebase Realtime Database. Only moderators with access to the Extension can read or write this data. No personal user data is stored there.

3.4 GitHub (update checks)

The Extension periodically checks raw.githubusercontent.com for a new version manifest to notify you of available updates. No user data is sent in this request.

4. Permissions Justification

  • storage — to save your settings across sessions and devices
  • alarms — to schedule daily update checks in the background
  • Host permissions (mangalib.me, ranobelib.me, etc.) — to inject the Extension's UI into the supported sites
  • Host permissions (AI APIs) — to send analysis requests to your chosen AI provider

5. Children's Privacy

The Extension is intended for adult moderators of Lib-family sites. We do not knowingly process data from children under 13.

6. Changes to This Policy

We may update this policy as the Extension evolves. The updated date at the top of this page reflects the latest revision.

7. Contact

If you have questions about this Privacy Policy, please open an issue on the GitHub repository.
