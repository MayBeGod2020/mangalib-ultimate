# Mangalib Ultimate Helper

Расширение для модераторов и редакторов сайтов семейства Lib: MangaLib, RanobeLib, HentaiLib, AnimeLib, SlashLib.

Активируется **только** для пользователей с ролью модератора или администратора — обычные пользователи не увидят никакого интерфейса.

---

## Возможности

- **🛡️ Модерация** — быстрый доступ к попапу бана, история банов, пакетное удаление комментариев, фильтрация по категориям нарушений
- **🤖 ИИ-анализ** — автоматическая проверка комментария на нарушение правил через выбранный AI-провайдер (DeepSeek, ChatGPT, Gemini, Claude и др.)
- **📊 Командная панель** — онлайн-статус модераторов, watchlist ключевых слов, объявления команды в реальном времени
- **👤 Тултип пользователя** — при наведении на ник показывает уровень, роли, дату регистрации и активные баны
- **📖 Автоскролл** — плавный скролл при чтении манхвы с автопереходом на следующую главу
- **🎨 Персонализация** — пресеты тем, кастомный цвет акцента, обои, шрифт и межстрочный интервал для ранобе

## Установка

### Chrome Web Store / Firefox Add-ons

*(ссылки появятся после публикации)*

### Вручную — Chrome / Edge / Opera / Yandex Browser

1. Скачай [последний релиз](https://github.com/MayBeGod2020/mangalib-ultimate/releases) и распакуй zip
2. Открой страницу расширений:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Opera: `opera://extensions`
   - Yandex: `browser://extensions`
3. Включи **«Режим разработчика»**
4. Нажми **«Загрузить распакованное расширение»** и выбери папку

### Вручную — Mozilla Firefox

1. Скачай [последний релиз](https://github.com/MayBeGod2020/mangalib-ultimate/releases) (zip — без распаковки)
2. Открой `about:debugging` → **«Этот Firefox»** → **«Загрузить временное дополнение»**
3. Выбери скачанный zip-файл

> Firefox и Chrome используют **один и тот же zip** — отдельных сборок не нужно.

## Использование

После установки на сайтах семейства Lib (при наличии роли модератора) в правом верхнем углу появятся кнопки:

- **📊 Панель** — командная панель модераторов
- **⚙️** — настройки расширения

Все функции включаются и настраиваются через шестерёнку.

---

## Настройка ИИ-анализа

1. Открой ⚙️ → вкладка **«🤖 ИИ»**
2. Выбери провайдера (DeepSeek, ChatGPT, Gemini, Claude, Qwen, Grok, Mistral)
3. Вставь API-ключ от выбранного провайдера
4. Ключ хранится локально в браузере и отправляется только выбранному провайдеру

## Горячие клавиши (попап бана)

| Комбинация | Действие |
|---|---|
| `Ctrl+Enter` | Подтвердить бан |
| `Escape` | Закрыть попап |
| `Ctrl+B` | Чекбокс «Забанить» |
| `Ctrl+P` | Чекбокс «Перманентный» |
| `Ctrl+1`–`9` | Выбрать причину бана |
