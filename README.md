# Mangalib Ultimate Helper

Объединённое расширение для модераторов сайтов семейства Lib. Включает:

- **🛡️ Модерация** — автозаполнение попапов бана, шпаргалка правил, фильтры карточек
- **📊 Командная панель** — список модераторов онлайн, watchlist, объявления
- **📖 Чтение** — автоскролл при чтении манхвы
- **🎨 Персонализация** — темы, обои, шрифты для глав ранобе

---

## Установка

### Chrome / Edge / Opera / Opera GX / Yandex Browser

1. Откройте страницу расширений:
   - Chrome:  `chrome://extensions`
   - Edge:    `edge://extensions`
   - Opera:   `opera://extensions`
   - Yandex:  `browser://extensions`

2. Включите **"Режим разработчика"**

3. Нажмите **"Загрузить распакованное расширение"**

4. Выберите папку `mangalib-ultimate`

### Mozilla Firefox

В `manifest.json` замените:
```json
"manifest_version": 3
```
На:
```json
"manifest_version": 2
```

И блок `background`:
```json
"background": {
    "service_worker": "background.js"
}
```
На:
```json
"background": {
    "scripts": ["background.js"],
    "persistent": false
}
```

Затем `about:debugging` → "Этот Firefox" → "Загрузить временное дополнение"

## Использование

После установки в правом верхнем углу появятся две кнопки:
- **📊 Панель** — командная панель модераторов (только для модераторов)
- **⚙️** — настройки расширения

Все функции включаются и настраиваются через шестерёнку.

---

## Оптимизация запросов

Расширение бережёт квоту API:
- Список модераторов — кэш 24 часа в `chrome.storage.local`
- Онлайн статусы — настраиваемый интервал (5/15/30 минут)
- Запросы по одному с задержкой 2 секунды между ними
- Тайтлы кэшируются в `sessionStorage` на время сессии
- При закрытой панели Dashboard данные не обновляются

## Горячие клавиши (попап бана)

- `Ctrl+Enter` — подтвердить бан
- `Escape` — закрыть попап
- `Ctrl+B` — чекбокс "Забанить"
- `Ctrl+P` — чекбокс "Перманентный"
- `Ctrl+1..9` — выбрать причину бана
