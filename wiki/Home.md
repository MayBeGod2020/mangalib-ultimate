# 🛡️ Mangalib Ultimate Helper — Полное руководство

Браузерное расширение для модераторов сайтов семейства Lib (MangaLib, AnimeLib, HentaiLib, RanobeLib).

---

## 📦 Установка

### Chrome / Opera / Edge
1. Скачай [последний релиз](https://github.com/MayBeGod2020/mangalib-ultimate/releases) — файл `mangalib-ultimate-*.zip`
2. Распакуй архив в любую папку
3. Открой `chrome://extensions/` → включи **Режим разработчика**
4. Нажми **Загрузить распакованное расширение** → выбери папку

### Firefox
1. Перейди на [Firefox Add-ons](https://addons.mozilla.org/) (поиск «Mangalib Ultimate»)
2. Или скачай `.xpi` из релизов и перетащи в Firefox

> ⚠️ Расширение активируется только для пользователей с ролью модератора или администратора.

---

## 🛡️ Модуль модерации

### Автозаполнение попапа бана
При клике «Удалить» / открытии попапа бана автоматически подставляются:
- Название тайтла
- Имя автора комментария
- Текст комментария
- Причина жалобы (если она была выбрана)

### Цветовая кодировка жалоб
Карточки жалоб окрашиваются по типу нарушения — сразу видно что где.

### Шпаргалка правил
В попапе бана под выбранной причиной появляется текст соответствующего правила.

### Автоскролл к следующей жалобе
После закрытия попапа страница автоматически прокручивается к следующей непросмотренной карточке.

### Автоотметка «Забанить»
При открытии попапа галочка «Забанить» ставится автоматически — одним кликом меньше.

## 🤖 ИИ-анализ нарушений

При открытии попапа бана ИИ анализирует комментарий и показывает:
- **Вердикт**: нарушает / не нарушает
- **Объяснение** на русском
- **Уверенность** в процентах (0–100%)

### Кнопка «🔍 Проверить страницу»
Пакетная проверка всех комментариев на странице. Подозрительные помечаются значком ⚠️.

### Настройка ИИ (подробнее — в [AI-Setup](AI-Setup))

**Самый простой способ (бесплатно, работает в РФ):**
1. Зайди на [console.groq.com](https://console.groq.com) → создай аккаунт
2. Нажми **Create API Key**, скопируй ключ
3. В настройках расширения (⚙️ → 🤖 ИИ) вставь ключ в поле **Groq API ключ**
4. Включи тумблер **ИИ анализ**

Также поддерживаются: DeepSeek, OpenAI, Gemini, Claude, Qwen, Grok, Mistral.

## 📊 Командная панель

Кнопка **📊 Панель** показывает:
- **Онлайн-статус** членов команды (обновляется каждые 5/15/30 мин)
- **Watchlist** ключевых слов — предупреждение при их появлении в комментариях
- **Объявления** для команды

Данные хранятся отдельно для каждого сайта (MangaLib, AnimeLib и т.д.).

## 🔔 Webhook-уведомления

При высокой уверенности ИИ (≥80%) отправляется push-уведомление.

### ntfy.sh (работает в РФ без VPN)
```
https://ntfy.sh/твоя-тема
```
1. Установи приложение ntfy на телефон
2. Подпишись на свою тему
3. Вставь URL в поле Webhook в настройках

### Telegram
```
https://api.telegram.org/bot<TOKEN>/sendMessage?chat_id=<CHAT_ID>
```
1. Создай бота через [@BotFather](https://t.me/BotFather)
2. Получи TOKEN и CHAT_ID
3. Вставь URL в настройки

### Discord
Вставь URL вебхука канала напрямую.

## 🔧 Устранение неполадок

**Кнопка ⚙️ не появляется**
- Проверь, что у тебя есть роль модератора на сайте
- Перезагрузи страницу (F5)
- Отключи другие расширения, которые могут конфликтовать

**ИИ не отвечает**
- Проверь правильность API ключа
- Убедись, что тумблер «ИИ анализ» включён
- Groq: убедись, что не исчерпан лимит запросов (free tier: ~14K токенов/мин)

**Автозаполнение не работает**
- Убедись, что тумблер «Автозаполнение попапа бана» включён
- Попробуй отключить и включить модуль модерации

**Webhook не приходит**
- Проверь URL (для ntfy.sh: `https://ntfy.sh/ТВОЯ_ТЕМА`)
- ИИ отправляет уведомление только при уверенности ≥80%
