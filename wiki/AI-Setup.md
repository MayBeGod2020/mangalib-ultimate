# 🤖 Настройка ИИ-анализа

Подробное руководство по настройке ИИ-анализа нарушений в Mangalib Ultimate Helper.

---

## ⚡ Быстрый старт (бесплатно, 2 минуты)

Используй **Groq** — это бесплатный провайдер с быстрым Llama 3. Работает в России без VPN.

1. Зайди на [console.groq.com](https://console.groq.com)
2. Зарегистрируйся через Google или GitHub аккаунт
3. Нажми **Create API Key** → дай имя ключу → **Submit**
4. Скопируй ключ (начинается на `gsk_...`)
5. Открой расширение: **⚙️ → вкладка 🤖 ИИ**
6. Вставь ключ в поле **«Groq API ключ»** → **Сохранить**
7. Включи тумблер **«ИИ анализ комментариев»**

Готово! ИИ будет анализировать каждый комментарий при открытии попапа бана.

## ⚙️ Порядок приоритетов

1. Если заполнен **основной API ключ** → используется выбранный провайдер
2. При ошибке 402 (оплата) или 429 (лимит) → автоматически переключается на **Groq**
3. Если основной ключ пустой, но заполнен Groq → **Groq работает как основной**

## 📝 Кастомные правила форума

По умолчанию ИИ использует встроенные правила MangaLib. Ты можешь задать свои:

1. ⚙️ → 🤖 ИИ → раздел **«Правила форума для ИИ»**
2. Напиши правила в формате:
```
1. Оскорбления — запрещены нападки на пользователей, авторов, персонажей
2. Спойлеры — запрещено раскрывать сюжет без предупреждения
3. Реклама — запрещены ссылки на сторонние ресурсы
```
3. Нажми **«Сохранить правила»**

Если поле пустое — используются правила по умолчанию.

## 🔍 Кнопка «Проверить страницу»

Пакетный режим — проверяет все комментарии на странице за один запуск.

**Включение:**
- ⚙️ → 🤖 ИИ → тумблер **«Кнопка сканирования страницы»** → ВКЛ

**Использование:**
1. Открой страницу со списком жалоб / комментариев
2. Нажми **🔍 Проверить страницу** (кнопка появляется рядом с ⚙️)
3. Дождись завершения (прогресс-бар)
4. Карточки с нарушениями будут помечены ⚠️ с уровнем уверенности

## 🔔 Webhook при высокой уверенности

Когда ИИ уверен ≥80% что комментарий нарушает правила — отправляется уведомление.

### ntfy.sh (рекомендуется для РФ)
**Преимущества:** бесплатно, работает без VPN, есть приложение для iOS и Android.

1. Установи [приложение ntfy](https://ntfy.sh/) на телефон
2. Придумай уникальное название темы, например `moders-mangalib-ivanov`
3. Подпишись на эту тему в приложении
4. В настройках расширения вставь: `https://ntfy.sh/moders-mangalib-ivanov`

### Telegram Bot
1. Открой [@BotFather](https://t.me/BotFather) → `/newbot`
2. Получи **TOKEN** вида `1234567890:AAA...`
3. Узнай свой **CHAT_ID**: напиши боту, потом открой `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. В настройках вставь:
```
https://api.telegram.org/bot<TOKEN>/sendMessage?chat_id=<CHAT_ID>
```

### Discord Webhook
1. Настройки канала → Интеграции → Вебхуки → Создать вебхук
2. Скопируй URL и вставь в настройки расширения

## 🛠️ Диагностика

**Вердикт не появляется в попапе**
- Убедись что тумблер «ИИ анализ» включён
- Проверь что ключ сохранён (должна быть надпись «✓ Ключ сохранён»)
- Открой DevTools (F12) → Console — ищи ошибки с префиксом `[MU:AI]`

**Ошибка 401 (Unauthorized)**
- Ключ неверный или истёк — создай новый

**Ошибка 429 (Rate limit)**
- Превышен лимит запросов — Groq автоматически подхватит как фолбэк
- Или подожди 1 минуту

**Ошибка 402 (Payment required)**
- Кончились деньги на балансе основного провайдера
- Расширение автоматически переключится на Groq (если ключ задан)

**Groq недоступен**
- Проверь наличие интернета
- Попробуй зайти на console.groq.com — возможны технические работы

---

## 🔒 Безопасность

- API ключи хранятся **только локально** в `chrome.storage.local` твоего браузера
- Ключи никогда не передаются на серверы расширения (у расширения нет своих серверов)
- Запросы отправляются напрямую выбранному провайдеру ИИ
- Текст комментария передаётся провайдеру для анализа — учти это при работе с чувствительным контентом
